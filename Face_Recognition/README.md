# Face Recognition Using CNN

## Overview

This project implements a Face Recognition system using a Convolutional Neural Network (CNN). The model is trained on the Labeled Faces in the Wild (LFW) dataset to identify different individuals from facial images.

## Dataset

Dataset Name: Labeled Faces in the Wild (LFW)

Dataset Link:
https://vis-www.cs.umass.edu/lfw/

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* Scikit-learn
* Seaborn

## Features

* Face image preprocessing
* CNN model implementation
* Model training and evaluation
* Accuracy and loss graphs
* Confusion matrix
* Classification report
* Model saving and loading

## Results

The model achieved high training accuracy and demonstrated good performance on unseen test images.
